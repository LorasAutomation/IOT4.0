Start Up Of Production Process
Sl. NO Check Points Status Comment/Remarks
1 Check for Right Register Ring ( LOCATING RING ) & Tie Bar is in Tac
2 Check Cooling Nipples / Hose Pipes connectors
3 Check for Hot oil Connections if required for the Mould & Check the Connected Cooling lines put on the Water and Check for any Leakage
4 Check for Mould Surface (core & cavity) for any oil / grease and make sure the Pillars and Bushes are Lubricated
5 Check the Ejector System ( spring tension if required )
6 Check if there is any Hydraulic Core system is Required for the Mould (If required make the hydraulic connections)
7 Check the Mould safety & set the required tonnage
8 Check for any Hot sprue / Hot runners available in the mould
9 If Hot runner systems are available Switch on the heater controller and set 5096 the MPD value AND KEEP CLOSE THE MOULD . (After all settings before starting set the temp as per MPD and wait for soaking)
10 Check the sprue bush / nozzle radius for butting for any leakages
11 Check and confirm the right material (GRADE) is available in the hopper / oven
12 Check the raw material wether Pre heated as per MPS and ensure that hopper / oven in working condition
13 Check for barrel is Cleaned with suitable material before starting with Right material
14 Check for Right material to start Production
15 Check the parameters as per Mould Process Data (MPS)
16 Srart the process and Check five consecutive parts for visual defects ( with ref. to reference samples )
17 Check the part? If it is OK offer it to Quality for Dimentional Checks
18 If the part is not ok fine tune the process parameters within the tolerance. In case of abnormality contact your superiors
19 After fine tuning if the part is ok then re offer to Quality for Dimention checks if part is ok to continue production
20 The initial start up parts must be destroyed (cut into pieces) to avoid mix up
